Gallery frame sizes

Auto Let the gallery choose a balanced frame based on sequence.
Standard Compact 2-column frame. Good for ordinary landscape or detail shots.
Wide 3 columns x 1 row on desktop. Best for cinematic horizontal photos.
Tall 2 columns x 2 rows on desktop. Best for vertical or rotated portraits.
Feature 3 columns x 2 rows on desktop. Use sparingly for standout images.
